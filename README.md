# Chef's Cuisine

레시피를 공유하고 관리할 수 있는 웹 플랫폼 팀 프로젝트입니다.  
사용자는 레시피를 등록하고, 이미지를 업로드하며, 다른 사용자와 댓글 및 좋아요를 통해 소통할 수 있습니다.

## 프로젝트 소개

Chef's Cuisine은 요리 레시피를 쉽고 편리하게 공유하기 위한 웹 서비스입니다.  
레시피 목록 조회, 상세 조회, 이미지 등록, 댓글, 좋아요, 마이페이지 등의 기능을 구현했습니다.

## 담당 역할

본인은 프로젝트에서 일정관리 및 기능 구현을 담당했습니다.

### 주요 담당 업무

- 프로젝트 일정 관리
- 기능 구현 진행 상황 정리
- 프론트엔드 화면 구성 보조
- 백엔드 API 연동 확인
- 데이터베이스 연동 및 오류 확인
- 팀원 간 GitHub 작업 흐름 관리

## 프로젝트를 통해 배운 점

이번 프로젝트를 통해 프론트엔드와 백엔드가 서로 데이터를 주고받는 흐름을 이해할 수 있었습니다.  
또한 GitHub를 활용한 협업 방식, 브랜치 관리, 충돌 해결, 일정 관리의 중요성을 경험했습니다.

특히 팀 프로젝트에서는 기능 구현뿐만 아니라 팀원 간 진행 상황을 공유하고, 오류를 함께 해결하는 과정이 중요하다는 것을 배웠습니다.
